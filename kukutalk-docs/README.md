# KUKUTalk — Pakiet dokumentów (bez kodu)
**Data pakietu:** 2025-08-26

Ten pakiet nie zawiera *kodu źródłowego*. Zawiera natomiast komplet plików:
- **SUPERPROMPT.md** – gotowa instrukcja dla ChatGPT do wygenerowania monorepo (NestJS + Next.js + PostgreSQL + Redis + Docker + testy).
- **INSTALL.md** – instrukcja instalacji/uruchomienia środowiska (Docker i lokalnie).
- **ARCHITECTURE.md** – opis architektury i komponentów (API, Web, mikroserwisy AI – stuby).
- **DOWNLOADS.md** – oficjalne linki do pobrania niezbędnych narzędzi (Docker Desktop, Node.js LTS, pnpm, Git, PostgreSQL, Redis, VS Code).
- **FILES_TO_GENERATE.md** – pełna lista struktury plików, które zostaną wygenerowane, gdy użyjesz SUPERPROMPT.md.
- **PRIVACY_SECURITY.md** – zasady bezpieczeństwa i prywatności (RODO-by-design, least-privilege, audyt).
- **CHECKLIST.md** – lista kontrolna „Go/No-Go” dla MVP.

## Jak użyć
1) **Otwórz `SUPERPROMPT.md` i wklej jego treść** do nowej rozmowy z ChatGPT (GPT-5 Thinking lub nowszy).  
2) Asystent wygeneruje cały kod i spakuje go w archiwum ZIP do pobrania.  
3) Po pobraniu kodu przejdź do **INSTALL.md**, aby uruchomić projekt.

> Pakiet jest samowystarczalny logistycznie – nie zawiera kodu, a jedynie instrukcje, plan i linki do pobrania wymaganych narzędzi.
